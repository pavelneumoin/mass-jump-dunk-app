# Масса + Данк

Локальное сайт-приложение для 8-недельного блока набора мышечной массы, уличных тренировок и увеличения прыжка.

Открой `index.html` в браузере. Данные чек-листов и замеров сохраняются в этом браузере через localStorage.

Важно: программа не заменяет врача. При боли в груди, головокружении, резкой боли в суставах, проблемах с давлением, почками или активном выпадении волос с воспалением кожи головы лучше пройти очную консультацию.
